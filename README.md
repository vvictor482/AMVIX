   AMVIX
É Projetada para otimizar a gestão fiscal e contábil de empresas por meio de recursos automatizados.

O sistema AMVIX foi projetado com uma arquitetura tecnológicacontemporânea e escalável,empregando ferramentas reconhecidas no mercado para assegurar um alto desempenho e facilidade de
manutenção. A parte do front-end foi desenvolvida utilizando React,TypeScript e Vite, tecnologias que oferecem uma interface responsiva e eficiente, além de tipagem estática, 
o que aumenta a segurança e a confiabilidade do código. 
Já o back-end foi estruturado com base nas soluções do Supabase, que integra um banco de dados PostgreSQL completo, com suporte a APIs automáticas, Realtime, Edge Functions e autenticação segura.
